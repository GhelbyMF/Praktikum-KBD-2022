# Praktikum-KBD-2022
Praktikan: Ilmu Kelautan 2021

## Pendahuluan

Repository ini digunakan untuk praktikum Komputer dan Big Data 2022, menggunakan python (versi 3 keatas), selain memahami syntax pada bahasa pemrograman python, mahasiswa mengerti konsep dari algoritma, logika dalam memecahkan masalah dan mengimplementasikan nya ke dalam bahasa pemrograman. 

## Tools (IDE, compiler)
Tools yang digunakan:
- [Python latest](https://www.python.org/downloads/) 
- [Spyder IDE](https://www.spyder-ide.org/)
